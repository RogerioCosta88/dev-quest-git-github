Configurando Usuário e Email no Git

Abrir o Git e digitar:

git config --global user.name "NOME DO USUÁRIO" apertar a tecla ENTER.

Para configurar o nome de usuário e para configurar o email:

git config --global user.email "EMAIL" apertar a tecla ENTER.

Obs.: é importante colocar o mesmo email cadastrado no GitHub.

Para verificar o que foi cadastrado, digitar:

git config --list apertar a tecla ENTER.

Esse comando lista todas as informações cadastradas.

Para buscar uma informação especifica:

git config user.name para buscar o nome de usuário cadastrado

git config user.email para buscar o email cadastrado
